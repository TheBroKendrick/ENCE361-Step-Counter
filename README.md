# Step Counter
### ENCE361 project

A step counter featuring the LSM6DS3TR-C Inertial Measurement Unit. 


**Instructions for use**
* Power the board on
* Start walking! Your steps will be displayed on the device's OLED display. The device will start to count successive steps after the 
  first two steps
* To enter test mode double tap the large bottom-most button
* To set the step goal, hold the joystick down. Use the potentiometer in the upper right corner to adjust the goal. To revert back to 
  the original goal, click the joystick. To confirm the new goal, hold the joystick down once again
* Move joystick left/right to change display states
* Move joystick up and down to change step units
